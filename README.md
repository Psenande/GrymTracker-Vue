# GrymTracker Vue (scaffold)

Proyecto scaffold para la práctica: Vue 3 + Vite + Pinia + Vue Router + Axios + Tailwind CSS.

Comandos:

```bash
npm install
npm run dev
```

Notas:
- Ajusta `src/api/axios.ts` para apuntar a la URL de la API real.
- Implementa las vistas y componentes adicionales según los requisitos de la práctica.
